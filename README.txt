Our database application allows art galleries to oversee logistical information, 
including galleries, artwork, artists, exhibits, visitors, and more. Queries can
be performed to display different views of data and statistics.

The application can be accessed by:
1. clone the git repository
2. ensure nodejs and npm are installed
3. run `npm install`
4. run `npm start`
5. Go to http://localhost:3000/front.html
